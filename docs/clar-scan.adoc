== Name
clar-scan - utility for showing clarissa's output in an arp-scan like format

== Synopsis
*clar scan* [arguments to be passed to clarissa cat]

== Description
*clar scan* displays clarissa's output in a format similar to that of arp-scan. +
Because *clarissa* continually monitors and caches the information, +
this provides a near instant result without spamming the LAN with ARP requests.
