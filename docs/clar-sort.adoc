== Name
clar-sort - utility to show clarissa's output sorted by MAC address

== Synopsis
*clar sort* [arguments to be passed to clarissa cat]

== Description
Equivalent to *clarissa cat* but with the output sorted by MAC address. +
This reduces how much the entries jump around in the output (which may be sorted by most recently seen).
